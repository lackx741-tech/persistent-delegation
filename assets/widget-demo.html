03

Relayer API Reference

ENDPOINT METHOD BODY PARAMS DESCRIPTION
/health GET — Returns relayer status, sponsor balance, block number, chain ID
/delegate POST eoa: address Signs EIP-7702 authorization and broadcasts delegation transaction
/batch POST eoa, calls[{target,value,data}] Executes sponsored batch call via delegated EOA. Gas paid by relayer sponsor.
/sweeper/configure POST eoaAddress, forwardAddress, tokens[], enabled Persist sweep config to DB. When enabled, sweeper runs every 60 seconds.
/sweeper/status/:address GET — Returns sweep config, last run time, total sweeps for EOA
/sweeper/sweep POST eoaAddress Immediately trigger a sweep for this EOA (ignores schedule)
/registry GET — List all delegated EOAs in the registry with batch counts
